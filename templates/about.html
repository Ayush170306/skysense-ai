About SkySense

🌦 Our Mission

SkySense aims to provide intelligent, accessible, and real-time weather insights using AI-powered forecasting models. It helps users make informed decisions with accurate weather prediction and easy-to-use tools.

🛠 Built With
  • Python, Flask
  • HTML, CSS, JavaScript, Bootstrap, Plotly.js
  • LSTM model (TensorFlow/Keras)
  • IBM Watson Studio (Model Training)
  • IBM Watson Assistant (Chatbot)
  • OpenWeatherMap API
